# Chart.js — First Chart Only (Grouped bars per state)

- States ordered with **largest funding first**
- State label centered on each bar pair (one y-label per state)
- Per state: **FY2017–FY2020 avg (light)** first, then **FY2024 (solid)**

White background, no x-axis ticks/labels/verbiage. Party color by 2020 winner.

Run:
```bash
npm install
npm run dev
# open http://localhost:8080
```
